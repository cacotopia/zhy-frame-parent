# frame

#### 介绍

分布式后台常用技术完全整合架构。通过maven依赖简单的引入即可使用各种常用技术。

#### 使用方式

通过maven依赖的方式引用到项目模块即可。把frame作为所有项目的parent，子项目一般都要引入common，其他模块根据系统需要引入，详见各个module的readme.md。
